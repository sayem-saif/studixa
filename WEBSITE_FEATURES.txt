STUDIXA WEBSITE FEATURE INVENTORY

Source reviewed:
- Live site at http://localhost:8080/
- Main app routing and page components
- School and college dashboard components
- Landing page sections and feature cards

1) Public / Marketing Website
- Home page with navbar, hero section, feature section, pricing section, about section, and footer.
- Main headline and positioning around AI-powered learning for school and college students.
- CTA buttons to sign up / log in and to explore features.
- Public statistics shown on the landing page:
  - 50+ chapters
  - AI powered
  - 500+ quizzes
  - 10K+ students
- Feature highlights shown on the landing page:
  - AI Mentor
  - Smart Summarizer
  - Daily Quizzes
  - Certificate Tracker
  - Event Notifications
  - Skill Courses
- Pricing plans:
  - Freemium plan
  - Premium plan at INR 299/month
- About section with founder/team story and mission.

2) Authentication and Account Flow
- Sign up and sign in on the auth page.
- Email verification for new accounts.
- Password visibility toggle.
- Redirect logic after login:
  - New users go to onboarding
  - Existing users go to the correct dashboard based on profile type
- Toast messages for common auth errors.

3) Onboarding Flow
- User chooses education level:
  - School student
  - College student
- School onboarding:
  - Select class 9, 10, 11, or 12
  - For class 11 and 12, select stream:
    - PCM
    - PCB
- College onboarding:
  - Select course
  - Select branch
- Saves profile data to Supabase and routes users to the right dashboard.

4) School Dashboard Features
- Personalized header for the logged-in student.
- Tabs for:
  - Subjects
  - AI Mentor
  - Document Summarizer
  - Daily Quiz
  - Progress
- Subject browsing by class and stream.
- Chapter browsing inside each subject.
- Chapter-level quizzes.
- Full class quiz mode.
- AI mentor for study help and guidance.
- Document summarizer for study material.
- Progress tracking for chapters, quizzes, scores, and streaks.
- Logout support.

5) School Learning Tools
- Subjects grid based on class and stream.
- Chapter view for each subject.
- Quiz section with:
  - Random question selection
  - Answer checking
  - Explanations
  - Score summary
  - Quiz retry
- Progress tracker with:
  - Chapters completed
  - Quizzes taken
  - Average score
  - Study streak
  - Subject-wise progress bars

6) College Dashboard Features
- Personalized header for the logged-in college student.
- Dashboard sections for:
  - CGPA Tracker
  - Certificate Tracker
  - Notifications
  - Skill Courses
  - Internship Application
  - AI Mentor
  - Document Summarizer
  - Project Tracker
- Feature cards on the main dashboard.
- Quick actions for CGPA, certificates, and learning.
- Internship application entry point.
- Logout support.

7) College Learning and Career Tools
- CGPA tracker:
  - Add semester records
  - Edit semester records
  - Delete semester records
  - Calculate current CGPA
- Certificate tracker:
  - Add certificates
  - Categorize certificates
  - Store optional certificate links
  - Delete certificates
- Notifications panel:
  - Hackathons
  - Internship opportunities
  - Event links and quick links to external platforms
- Skill courses:
  - C programming
  - Python
  - Java
  - Web development
- Internship application module.
- AI mentor for college guidance.
- Document summarizer for PDFs and study material.
- Project tracker for GitHub projects.

8) Certificate Verification
- Public certificate verification page at /verify/:certificateNumber.
- Looks up certificate data in Supabase.
- Shows verified certificate details if valid.
- Shows not found message if invalid.

9) App / Platform Infrastructure
- React Router based navigation.
- Supabase authentication and database integration.
- Toast notifications.
- Animated UI transitions using Framer Motion.
- Shared UI components such as buttons, inputs, selects, badges, dialogs, tooltips, and toasters.
- Distinct routing for school and college users.

10) Overall Product Summary
- Studixa is an AI-powered education platform for both school and college students.
- The platform combines learning content, quizzes, AI assistance, progress tracking, certificate management, career opportunities, and internship support in one app.
